Как играть в Колор Дайс

Чтобы начать играть в Колор Дайс, достаточно придумать шесть действий, связать их с цветами и бросить от одного до шести онлайн-кубиков. Подготовка занимает около пяти минут, а продолжительность партии определяют сами участники.

Открыть Колор Дайс

Пять шагов до первого броска

  1. 1

    Определите цель игры

    Решите, что будет происходить после броска. Цвет может выбирать задание, упражнение, тему разговора, материал для рисунка, роль участника или направление следующего хода. Одна ясная цель делает правила понятными уже в первом раунде.

  2. 2

    Назначьте значения цветов

    Запишите действия для красного, оранжевого, жёлтого, зелёного, синего и фиолетового. Участники должны увидеть или услышать значения до броска. Можно взять готовую схему со страницы правил Колор Дайс.

  3. 3

    Выберите количество кубиков

    Один кубик подходит для одного действия. Два или три создают короткую последовательность, а четыре-шесть помогают распределить роли или собрать цветовую палитру. Выберите число в верхнем поле игрового блока.

  4. 4

    Сделайте бросок

    Нажмите «Бросить снова» для мгновенного результата. Мега-бросок открывает цвета по очереди, а скретч-карта позволяет стирать покрытие пальцем или мышью. Во всех режимах используются одинаковые шесть вариантов.

  5. 5

    Выполните результат

    Примените действие, которое было назначено выпавшему цвету. Если кубиков несколько, выполняйте их слева направо либо распределите по одному между игроками. Затем передайте ход и начинайте следующий раунд.

Готовый сценарий для компании

Для первой партии используйте один кубик и шесть простых действий. Красный означает «расскажи короткую историю», оранжевый - «выбери следующего игрока», жёлтый - «задай вопрос», зелёный - «получи одно очко», синий - «брось ещё раз», фиолетовый - «придумай общее задание».

Выберите порядок ходов. Первый игрок делает бросок в Колор Дайс и выполняет действие. После этого ход переходит по кругу. Игра заканчивается, когда каждый участник сделал одинаковое количество бросков. Побеждает игрок с наибольшим числом очков, если в правилах есть начисление баллов; без счёта партия завершается общей финальной задачей.

Как играть на уроке

Учитель заранее связывает цвета с учебными форматами. Например: красный - исправить ошибку, оранжевый - привести пример, жёлтый - выбрать ответ, зелёный - объяснить правило, синий - задать вопрос однокласснику, фиолетовый - выполнить творческую часть. Содержание меняется под предмет, а механика остаётся прежней.

Для общего класса удобно бросать один кубик и давать одно задание всем. Для работы по группам выберите столько кубиков, сколько команд участвует. Результаты читаются слева направо: первая команда получает первый цвет, вторая - второй и так далее. Если требуется одинаковая сложность, заранее подготовьте действия сопоставимого объёма.

Как использовать Колор Дайс для творчества

Выберите три кубика и назначьте им роли: основной цвет, дополнительный цвет и акцент. После броска соберите рисунок, коллаж, образ персонажа или оформление предмета из полученной палитры. Совпадение двух цветов означает, что оттенок занимает больше места в работе.

Для более сложного задания добавьте четвёртый кубик как цвет фона, пятый - как деталь и шестой - как ограничение. Например, фиолетовый на шестом кубике может означать геометрический узор, а зелёный - природный элемент. Значения лучше записать заранее, чтобы результат оставался случайным.

Как считать очки и завершить игру

Колор Дайс не навязывает систему победы. Для соревновательной партии назначьте каждому цвету количество очков или условие их получения. Простой вариант: успешное выполнение любого действия приносит одно очко, отказ не приносит баллов, а фиолетовый даёт бонусное очко. Все игроки должны сделать одинаковое количество бросков.

Для кооперативной игры установите общую цель, например набрать десять успешных действий до трёх красных результатов. Для спокойного занятия можно вообще не считать очки: договоритесь о пяти раундах или завершите партию после того, как выпадут все шесть цветов.

Что делать с повторяющимися цветами

Повторы допустимы и подтверждают независимость кубиков. До начала игры выберите один из трёх вариантов: выполнить действие несколько раз, распределить одинаковое действие между участниками или перебросить только повторяющиеся кубики. Не меняйте правило после появления комбинации, иначе участники будут по-разному оценивать одинаковые результаты.

Можно начинать

Подготовьте шесть значений, выберите участников и откройте Колор Дайс онлайн. Первый бросок займёт несколько секунд. Если нужен подробный справочник по значениям, количеству кубиков и режимам, вернитесь к странице «Правила Колор Дайс».
