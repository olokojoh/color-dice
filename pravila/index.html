Правила Колор Дайс: как читать цвета

Перед игрой назначьте каждому цвету действие. Затем выберите количество кубиков и примените выпавший результат. В Колор Дайс шесть цветов с одинаковой вероятностью. Эти правила подходят для одного игрока, команды или компании.

Бросить цветные кубики

Короткие правила Колор Дайс

  1. Определите цель. Кубик может выбирать действие, игрока, тему, материал, упражнение или часть задания.
  2. Назначьте значения. Свяжите шесть вариантов с красным, оранжевым, жёлтым, зелёным, синим и фиолетовым.
  3. Выберите кубики. Один кубик даёт один ответ. Несколько кубиков составляют последовательность.
  4. Сделайте бросок. Откройте Колор Дайс онлайн и выберите обычный бросок, мега-бросок или скретч-карту.
  5. Примените результат. Выполните действие выпавшего цвета. Меняйте правила только перед новым раундом и сразу для всех игроков.

Стандартные значения шести цветов

Колор Дайс не назначает значения автоматически. Их можно менять под урок, тренировку, настольную игру или творческое занятие. Если своих правил пока нет, возьмите эту схему.

Сколько кубиков выбрать

1 кубик

Используйте его для одного решения: действия, цвета команды, следующего игрока или типа задания.

2-3 кубика

Читайте их как короткую последовательность. Например, первый цвет задаёт тему, второй действие, а третий сложность.

4-6 кубиков

Такой набор подходит для палитры, распределения ролей или одновременного выбора для нескольких игроков.

Каждый кубик в Колор Дайс выбирает цвет независимо. Поэтому два зелёных кубика подряд являются обычным результатом. По правилам можно выполнить действие дважды, дать одну роль двум участникам или перебросить совпадение.

Правила трёх режимов

Обычный бросок

Нажмите «Бросить снова», чтобы открыть все цвета одновременно. Следующий бросок можно сделать той же кнопкой, без перезагрузки страницы.

Мега-бросок

В мега-режиме кубики открываются по очереди. Вероятности не меняются. Этот вариант подойдёт для финала или жеребьёвки, где каждый цвет нужно показать отдельно.

Скретч-карта

Скретч-карта скрывает каждый результат под серебристым покрытием. Сотрите часть слоя пальцем или мышью, и покрытие исчезнет полностью. С клавиатуры карта открывается клавишей Enter или пробелом.

Три готовых варианта правил

Для компании

Назначьте цветам короткие действия, например ответ на вопрос, историю, выбор следующего игрока, бонус, пропуск хода и повторный бросок. Участник выполняет действие и передаёт ход дальше.

Для урока

Свяжите цвета с учебной работой: чтением, письмом, обсуждением, рисунком, движением и проверкой ответа. Один кубик выбирает общее задание, несколько составляют разминку по шагам.

Для творческой палитры

Выберите три или четыре кубика. Первый результат станет основным цветом, второй дополнительным, а остальные акцентами. Повтор может сделать оттенок главным. Если это не подходит, заранее разрешите перебрасывать совпадения.

Что записать перед началом

На одном листе укажите значение каждого цвета, порядок чтения кубиков, правило для повторов и момент окончания партии. Если есть очки, запишите способ их начисления и одинаковое число ходов для каждого участника. Для командной игры сразу решите, кто получает первый, второй и последующие кубики.

Проведите один пробный бросок без очков. Если действие нельзя понять без спора или оно занимает заметно больше времени, чем остальные, исправьте формулировку до первого раунда. После начала игры меняйте правило только по общему решению и со следующего броска.

Начните первый раунд

Откройте генератор Колор Дайс, выберите количество кубиков и сделайте бросок. Подготовка участников, подсчёт очков и завершение партии описаны в инструкции «Как играть в Колор Дайс».
