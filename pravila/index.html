Правила Колор Дайс: как читать цвета

Главное правило Колор Дайс простое: заранее назначьте каждому цвету действие, выберите количество кубиков и выполните результат броска. В стандартном наборе шесть равновероятных цветов. Играть можно одному, по командам или всей компанией.

Бросить цветные кубики

Короткие правила Колор Дайс

  1. Определите цель. Решите, что будет выбирать кубик: действие, игрока, тему, материал, упражнение или часть задания.
  2. Назначьте значения. Сопоставьте шесть вариантов с красным, оранжевым, жёлтым, зелёным, синим и фиолетовым цветами.
  3. Выберите кубики. Один кубик даёт один ответ. Несколько кубиков создают последовательность или комбинацию.
  4. Сделайте бросок. Откройте Колор Дайс онлайн и нажмите кнопку обычного, мега- или скретч-режима.
  5. Следуйте результату. Выполните действие по выпавшему цвету. Перед следующим раундом правила можно оставить или изменить для всех игроков.

Стандартные значения шести цветов

У Колор Дайс нет единственного обязательного словаря. Это делает игру универсальной: один и тот же генератор подходит для урока, тренировки, настольной игры и творческого занятия. Если собственных правил пока нет, начните со следующей базовой схемы.

КрасныйСтоп или сложный ходПропустить ход либо выполнить самое трудное задание

Эти обозначения являются стартовым шаблоном, а не ограничением. Важно только согласовать значения до броска. Если менять смысл после того, как цвет уже выпал, случайный выбор перестаёт быть честным для участников.

Сколько кубиков выбрать

1 кубик

Подходит для одного решения: выбрать действие, цвет команды, следующего игрока или тип задания.

2-3 кубика

Создают короткую последовательность. Например, первый цвет задаёт тему, второй - действие, третий - сложность.

4-6 кубиков

Нужны для палитры, распределения ролей, длинного маршрута или одновременного выбора для нескольких игроков.

Каждый кубик в Колор Дайс выбирает цвет независимо. Совпадения разрешены: если два кубика стали зелёными, это обычный случайный результат. Можно выполнить действие дважды, дать одинаковую роль двум участникам или заранее договориться перебрасывать повторы.

Правила трёх режимов

Обычный бросок

Нажмите «Бросить снова», чтобы открыть все цвета одновременно. Это основной режим для быстрых раундов. После результата кнопка остаётся доступной, поэтому следующая комбинация появляется без перезагрузки страницы.

Мега-бросок

В мега-режиме кубики открываются по очереди. Вероятности остаются такими же, меняется только подача результата. Используйте этот вариант для финала, жеребьёвки или момента, когда участникам важно увидеть каждый цвет отдельно.

Скретч-карта

Скретч-режим скрывает каждый результат под серебристым покрытием. Сотрите слой пальцем или мышью. Карта считается открытой, когда видна достаточная часть цвета; после этого покрытие исчезает полностью. На клавиатуре результат можно открыть клавишей Enter или пробелом.

Три готовых варианта правил

Для компании

Назначьте цветам шесть коротких действий: ответить на вопрос, рассказать историю, выбрать другого игрока, получить бонус, пропустить ход или бросить ещё раз. Участники делают по одному броску, выполняют выпавшее действие и передают ход по кругу.

Для урока

Свяжите цвета с форматами работы: прочитать, написать, обсудить, нарисовать, показать жестом и проверить ответ. Учитель выбирает один кубик для общего задания или несколько кубиков для последовательной мини-разминки.

Для творческой палитры

Выберите три или четыре кубика. Первый результат используйте как основной цвет, второй - как дополнительный, остальные - как акценты. Если цвета повторились, сделайте повтор доминирующим оттенком или перебросьте только повторяющийся кубик по заранее выбранному правилу.
