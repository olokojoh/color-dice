Контакты

Об ошибке, проблеме с клавиатурой или неточности в правилах можно сообщить через открытый репозиторий Колор Дайс.

GitHub Issues

Открыть список обращений и создать новое

Обращения на GitHub видны всем. Не публикуйте там пароли, адреса, платёжные сведения и другие личные данные. У Колор Дайс нет аккаунтов, подписок и платных операций.

Что указать в сообщении

Для технической ошибки укажите страницу, браузер, устройство и действия перед сбоем. Для вопроса по правилам опишите игровой сценарий и количество кубиков. Скриншот можно приложить, если на нём нет личных данных.
