CONTACT THE PROJECT

Contact Color Dice

Report a bug, keyboard problem, or correction through the public Color Dice repository.

GitHub Issues

View existing reports or create a new issue

GitHub issues are public. Do not post passwords, addresses, payment details, or other personal information. Color Dice has no user accounts, subscriptions, or paid transactions.

What to include

For a technical problem, include the page, browser, device, and actions that led to the error. For a rules question, describe the game and number of dice. You may attach a screenshot if it contains no personal information.
