About Color Dice

Color Dice is an independent browser game that chooses random colors in the form of virtual dice. It is free to use for games, lessons, art prompts, and everyday choices.

Project and source code

The source code, change history, and issue tracker are public in the olokojoh/color-dice GitHub repository. The project is not a Google product and does not claim an official connection with another dice service.

How a roll works

The browser uses its cryptographic random number generator to choose each die separately. Red, orange, yellow, green, blue, and purple have the same chance. A color can appear more than once in a roll.

Local data

The selected theme, sound setting, roll count, and latest result stay in localStorage on the user's device. Color Dice does not create user accounts or send roll history to a server.
